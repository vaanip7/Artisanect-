# Artisanect

Artinect is an AI-powered marketplace designed to connect local artisans and handicraft sellers directly with customers through a modern digital platform.

## Features

- Artisan Registration & Profiles
- Product Listing & Management
- Product Search & Filtering
- Shopping Cart & Orders
- AI Product Description Generating
- AI Chatbot Assistant
- AI Language Translation


About Tech Stack
### Frontend
- React JS
- Bootstrap/Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### AI Integration
- Google Gemini API

## Project Goal

To empower artisans by providing a digital platform that expands market reach while preserving traditional craftsmanship.
